# poker-ui

Welcome to our poker UI!

## Getting started

Install our package
`npm i @cardano-after-dark/poker-ui`

Import our library
`import { CardValues, PokerCard, PokerTable, Suits } from '@cardano-after-dark/poker-ui';`

Use a component
`<PokerCard suit={Suits.Hearts} value={CardValues.King} />`
